Mathest API

Basic usage: This API is used to access the Drive sheets linked to the Mathest Cloud infrastructure.  The table below shows the endpoints and formats to use the api.

MethodAPI Endpoint Description
returnNextQuestion
 "?<base_url>/question?uid=<user_uid>&row=<next_row>&sno=<sheet_number>" Returns next question to be answered from the user database
returnNewSheet "<base_url>/sheet?uid=<user_uid>" Creates a new sheet and returns its ID for a new user
addition "<base_url>/addition?uid=<user_uid>&row=<\current_row>&answer=<answer_by_user>" Addition error classifier
subtraction "<base_url>/subtraction?uid=<user_uid>&row=<current_row>&answer=<answer_by_user>" Subtraction error classifier
multiplication "<base_url>/multiplication?uid=<user_uid>&row=<current_row>&answer=<answer_by_user>"

Multiplication error classifier
division
 "<base_url>/division?uid=<user_uid>&row=<current_row>&answer=<answer_by_user>"

Division error classifier
workingOfAddition
 "<base_url>/addition-working?uid=<user_uid>&number1=<first_number>&number2=<second_number>"

Shows working of Addition
workingOfMultiplication
 "<base_url>/multiplication-working?uid=<user_uid>&number1=<first_number>&number2=<second_number>"

Shows working of Addition
